먹히나?

Inline

  • 줄바꿈 없음
  • a, img, span
  • width, height로 크기 조절 X(=레이아웃에 활용 불가)
  • 좌우 margin, 좌우 padding 만 적용 가능
  • display:inline;

Block

  • 줄바꿈 있음
  • h, div, p
  • width, height로 크기 조절 가능
  • 내용과 별개로 자신의 영역을 가짐(내용을 삭제 해도 영역은 남아 있음
  • display:block;

inline-block

  • 줄바꿈 없음
  • block 요소 처럼 width, height 가짐
  • 상하 margin, padding도 존재
  • display:inline-block;
